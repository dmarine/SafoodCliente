## SafoodCliente start
```
git clone https://github.com/dmarine/SafoodCliente.git
```

### Instalar dependencias
```
npm install
```

### Build
```
npm run build
```

## Rutas
| Class        | Uri                            |
| ------------ | ------------------------------ |
| Home         | /                              |
| Foods        | /foods                         |
| Category     | /categorys/{id}                |
